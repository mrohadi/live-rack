## Inbound endpoints

| Connector | Path | Verification |
|---|---|---|
| Shopify | `/webhooks/shopify` | base64 HMAC-SHA256 of body |
| Square | `/webhooks/square` | HMAC over URL + body |
| Stripe | `/webhooks/stripe` | `Stripe-Signature` over `<t>.<body>` |
| Shippo | `/webhooks/shippo` | shared `X-Shippo-Token` |
| Billing | `/webhooks/billing` | Stripe signature → plan change |

All inbound bodies must be raw bytes — any proxy that reformats JSON breaks the
HMAC. Idempotency uses the vendor's event id, so safe to replay.

## Outbound
