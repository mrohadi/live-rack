## What is live-rack?

live-rack is a multi-tenant SaaS that connects your physical store layout, item
movement, POS sales, and external signals (weather, transit) into one
operational console.

## Architecture at a glance

React SPA + Scanner PWA → Go (Echo) API + WebSocket gateway → Postgres/Timescale
(OLTP) and ClickHouse (analytics), with NATS JetStream as the event bus. See
[Architecture](/essentials/architecture).
